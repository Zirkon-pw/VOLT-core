# Приложение А — Модули редактора

Каждая функция редактора — отдельный модуль в `kernel/editor/`.

## 1. `core/` — Ядро редактора

**Ответственность:** Создание и управление экземпляром Tiptap, конфигурация, жизненный цикл.

| Файл | Назначение |
|------|-----------|
| `EditorEngine.ts` | Фабрика экземпляров редактора. Создаёт, уничтожает, предоставляет доступ к экземплярам. |
| `EditorInstance.ts` | Обёртка над экземпляром Tiptap. Предоставляет методы: `getMarkdown()`, `setMarkdown()`, `focus()`, `save()`, `isDirty()`. |
| `EditorConfig.ts` | Конфигурация редактора: какие расширения включить, настройки readOnly, autofocus, placeholder. |

**Зависимости:** Tiptap core, расширения из `extensions/`, сериализация из `serialization/`.

## 2. `extensions/` — Расширения Tiptap

**Ответственность:** Каждое расширение Tiptap — отдельный модуль.

| Модуль | Назначение |
|--------|-----------|
| `slash-command/` | Slash-команды (`/` меню) — регистрация, рендер, выполнение |
| `code-block/` | Блок кода с подсветкой языка |
| `table/` | Таблицы с изменяемыми строками/столбцами, цветами ячеек |
| `math-block/` | Математические блоки (KaTeX) |
| `embed-block/` | Embed-блоки (YouTube, GitHub и т.д.) |
| `find-in-file/` | Подсветка совпадений поиска по файлу |

**Зависимости:** Tiptap core, каждый модуль независим.

## 3. `serialization/` — Сериализация Markdown

**Ответственность:** Конвертация между Prosemirror (внутреннее представление Tiptap) и Markdown.

| Файл | Назначение |
|------|-----------|
| `MarkdownParser.ts` | Парсинг Markdown → Prosemirror JSON. Использует `tiptap-markdown`. |
| `MarkdownSerializer.ts` | Сериализация Prosemirror JSON → Markdown. Использует `tiptap-markdown`. |

**Зависимости:** `tiptap-markdown`, конфигурация расширений.

## 4. `auto-save/` — Автосохранение

**Ответственность:** Тайминг автосохранения, отслеживание dirty-состояния, регистрация обработчиков сохранения.

| Файл | Назначение |
|------|-----------|
| `AutoSaveService.ts` | Сервис автосохранения. Запускает таймер, вызывает save handler при изменении. |
| `DirtyTracker.ts` | Отслеживание dirty-состояния. Сравнивает текущее содержимое с последним сохранённым. |

**Зависимости:** `EditorInstance` из `core/`.

## 5. `image-handling/` — Обработка изображений

**Ответственность:** Drag-drop изображений, вставка из буфера, разрешение изображений, конвертация base64.

| Файл | Назначение |
|------|-----------|
| `ImageDropHandler.ts` | Обработка drag-drop изображений в редактор. |
| `ImagePasteHandler.ts` | Обработка вставки изображений из буфера обмена. |
| `ImageResolver.ts` | Разрешение путей изображений (локальные файлы, base64, внешние URL). |

**Зависимости:** `EditorInstance` из `core/`, плагин `image-service` для операций с файлами.

## 6. `sessions/` — Управление сессиями редактора

**Ответственность:** Управление сессиями редактора, якоря, сессии плагинов.

| Файл | Назначение |
|------|-----------|
| `EditorSessionManager.ts` | Менеджер сессий. Регистрация, удаление, получение активной сессии. |
| `EditorSession.ts` | Сущность сессии. `getMarkdown()`, `save()`, `dispose()`, `getSelection()`, `createAnchor()`, `insertAtAnchor()`, `replaceRange()`. |
| `AnchorManager.ts` | Управление якорями. Создание, получение, удаление якорей. Bias start/end. |

**Зависимости:** `EditorInstance` из `core/`.

## 7. `toc/` — Оглавление

**Ответственность:** Генерация оглавления из заголовков документа.

| Файл | Назначение |
|------|-----------|
| `TocGenerator.ts` | Парсит markdown/Prosemirror, извлекает заголовки, строит дерево оглавления. |

**Зависимости:** `EditorInstance` из `core/` или сырой markdown.

## 8. `ui/` — UI-компоненты редактора

**Ответственность:** React-компоненты панели редактора.

| Файл | Назначение |
|------|-----------|
| `EditorPanel.tsx` | Контейнер редактора. Подключает все функции, управляет состоянием панели. |
| `MarkdownEditorSurface.tsx` | Поверхность рендера Tiptap. DOM-контейнер, в который монтируется Tiptap. |

**Зависимости:** Все модули editor, Zustand stores.

## 9. Граф зависимостей

```
ui/EditorPanel.tsx
├── core/EditorEngine.ts
│   ├── core/EditorConfig.ts
│   ├── core/EditorInstance.ts
│   ├── extensions/* (все расширения)
│   └── serialization/* (парсер + сериализатор)
├── auto-save/AutoSaveService.ts
│   └── auto-save/DirtyTracker.ts
├── image-handling/* (drop, paste, resolver)
├── sessions/EditorSessionManager.ts
│   ├── sessions/EditorSession.ts
│   └── sessions/AnchorManager.ts
└── toc/TocGenerator.ts
```
