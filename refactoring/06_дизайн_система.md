## 2. Правила UI-компонентов

Каждый компонент в `shared/ui/` **обязан**:

1. **Использовать только дизайн-токены** для цветов, отступов, типографики, border-radius, теней
2. **Без захардкоженных hex-значений** — использовать `var(--color-*)` токены
3. **Без произвольных значений px/rem** — использовать `var(--spacing-*)` токены
4. **Одинаковое фокусное кольцо** — `outline: 2px solid var(--color-focus-ring); outline-offset: 2px;`
5. **Одинаковые hover-состояния** — `background-color: var(--color-bg-hover); transition: all 0.15s ease;`
6. **Одинаковые disabled-состояния** — `opacity: 0.5; pointer-events: none;`
7. **Одинаковые размеры** — использовать `--spacing-lg` для padding, `--radius-sm` для border-radius

## 4. Линт-контроль

### 4.1 Stylelint

```javascript
// .stylelintrc.js
module.exports = {
  rules: {
    'color-no-hex': true,
    'custom-property-pattern': '^--(color|spacing|radius|shadow|z|font|text|leading|tracking)-',
    'declaration-property-value-disallowed-list': {
      'border-radius': ['^[0-9]+px$', '^[0-9]+rem$'], // Должны использоваться токены
      'padding': ['^[0-9]+px$', '^[0-9]+rem$'],
      'margin': ['^[0-9]+px$', '^[0-9]+rem$'],
      'color': ['^#[0-9a-fA-F]+$'],
      'background-color': ['^#[0-9a-fA-F]+$'],
    },
  },
};
```

### 4.2 ESLint (кастомный плагин eslint-plugin-volt)

Запрещает захардкоженные цвета в JS/TSX:
- Нет `color: '#239dad'` в inline-стилях
- Нет `backgroundColor: '#fdfcf8'` в объектах style
- Обязательно использовать CSS-токены или токены темы

```javascript
// .eslintrc.js
module.exports = {
  plugins: ['volt'],
  rules: {
    'volt/no-hardcoded-colors': 'error',
    'volt/no-hardcoded-spacing': 'error',
    'volt/no-hardcoded-radii': 'error',
  },
};
```

## 5. Паритет тем

Светлая и тёмная темы используют **идентичную структуру токенов**:

| Токен | Light | Dark |
|-------|-------|------|
| `--color-bg-primary` | `#fdfcf8` | `#161a1d` |
| `--color-bg-secondary` | `#f4f0e6` | `#1b2024` |
| `--color-text-primary` | `#1f2328` | `#e7ecef` |
| `--color-accent` | `#239dad` | `#39b6c5` |
| `--color-border` | `#d2cabd` | `rgba(231, 236, 239, 0.12)` |
| `--shadow-sm` | `0 1px 2px rgba(31, 35, 40, 0.04)` | `0 1px 2px rgba(0, 0, 0, 0.18)` |

Каждый токен присутствует в обеих темах. Нет токенов, существующих только в одной теме.

## 7. Структура каждого UI-компонента

Каждый компонент следует MVC-паттерну:

```

├── controller/
│   └── ButtonController.tsx   # Логика + рендер
├── model/
│   └── types.ts               # TypeScript типы
├── view/
│   ├── ButtonView.tsx         # Чистый view
│   └── ButtonView.module.scss # Стили (только токены)
└── index.ts                   # Баррель-экспорт
```
