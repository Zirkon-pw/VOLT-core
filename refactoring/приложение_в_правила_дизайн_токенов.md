# Приложение В — Правила использования дизайн-токенов

## 1. Таблица соответствия токенов и свойств

| Тип токена | Где использовать | Запрещено |
|-----------|-----------------|-----------|
| `--color-*` | Все значения color, background-color, border-color, fill, stroke | Захардкоженные hex/rgb/rgba |
| `--spacing-*` | Все padding, margin, gap, row-gap, column-gap | Захардкоженные px/rem |
| `--text-*` | Все font-size | Захардкоженные px/rem |
| `--leading-*` | Все line-height | Захардкоженные числа (кроме токенов) |
| `--tracking-*` | Все letter-spacing | Захардкоженные em |
| `--radius-*` | Все border-radius | Захардкоженные px/rem |
| `--shadow-*` | Все box-shadow | Захардкоженные строки теней |
| `--z-*` | Все z-index | Захардкоженные числа |
| `--font-*` | Все font-family | Захардкоженные названия шрифтов |

## 2. Примеры правильного использования

### 2.1 CSS Modules

```scss
// ButtonView.module.scss — ПРАВИЛЬНО
.button {
  padding: var(--spacing-sm) var(--spacing-lg);
  font-size: var(--text-sm);
  line-height: var(--leading-normal);
  color: var(--color-text-primary);
  background-color: var(--color-accent);
  border-radius: var(--radius-sm);
  border: 1px solid var(--surface-border-primary);
  box-shadow: var(--shadow-sm);
  z-index: var(--z-dropdown);

  &:hover {
    background-color: var(--color-accent-hover);
  }

  &:focus-visible {
    outline: 2px solid var(--color-focus-ring);
    outline-offset: 2px;
  }

  &:disabled {
    opacity: 0.5;
    pointer-events: none;
  }
}
```

### 2.2 Inline-стили в TSX

```tsx
// ПРАВИЛЬНО — использование CSS custom properties через style
<div style={{
  padding: 'var(--spacing-lg)',
  color: 'var(--color-text-primary)',
  backgroundColor: 'var(--color-surface-raised)',
} as React.CSSProperties}>
  Content
</div>
```

### 2.3 Tailwind-подобные утилиты (если используются)

```scss
// ПРАВИЛЬНО — утилиты на основе токенов
.text-primary { color: var(--color-text-primary); }
.bg-surface { background-color: var(--color-surface-raised); }
.p-lg { padding: var(--spacing-lg); }
.m-sm { margin: var(--spacing-sm); }
.rounded-sm { border-radius: var(--radius-sm); }
.shadow-md { box-shadow: var(--shadow-md); }
```

## 3. Примеры неправильного использования

### 3.1 CSS Modules

```scss
// ButtonView.module.scss — НЕПРАВИЛЬНО
.button {
  padding: 8px 16px;              // ❌ Захардкоженные px
  font-size: 14px;                // ❌ Захардкоженные px
  color: #1f2328;                 // ❌ Захардкоженный hex
  background-color: #239dad;      // ❌ Захардкоженный hex
  border-radius: 4px;             // ❌ Захардкоженные px
  box-shadow: 0 1px 2px rgba(0,0,0,0.1); // ❌ Захардкоженная тень
  z-index: 100;                   // ❌ Захардкоженное число
}
```

### 3.2 Inline-стили в TSX

```tsx
// НЕПРАВИЛЬНО — захардкоженные значения
<div style={{
  padding: '8px 16px',        // ❌
  color: '#1f2328',           // ❌
  backgroundColor: '#fdfcf8', // ❌
  borderRadius: '4px',        // ❌
}}>
  Content
</div>
```

## 4. Исключения

Следующие случаи **допустимы**:

| Исключение | Пример | Причина |
|-----------|--------|---------|
| `border: 1px solid` | `border: 1px solid var(--surface-border-primary)` | `1px` — допустим, это не отступ, а толщина линии |
| `opacity: 0.5` | `opacity: 0.5` | Прозрачность не токенизируется |
| `transition: all 0.15s ease` | `transition: all 0.15s ease` | Тайминги анимаций не токенизируются |
| `cursor: pointer` | `cursor: pointer` | Курсоры не токенизируются |
| `display: flex` | `display: flex` | Layout-свойства не токенизируются |
| `white-space: nowrap` | `white-space: nowrap` | Текстовые свойства не токенизируются |

## 6. ESLint плагин (eslint-plugin-volt)

```javascript
// eslint-plugin-volt/lib/rules/no-hardcoded-colors.js
module.exports = {
  meta: {
    type: 'problem',
    docs: {
      description: 'Запрещает захардкоженные цвета в inline-стилях и CSS-объектах',
    },
    messages: {
      noHardcodedColor: 'Используйте CSS custom property (var(--color-*)) вместо захардкоженного цвета',
    },
  },
  create(context) {
    return {
      JSXAttribute(node) {
        if (node.name.name === 'style' && node.value.expression.type === 'ObjectExpression') {
          node.value.expression.properties.forEach(prop => {
            if (prop.key.name === 'color' || prop.key.name === 'backgroundColor') {
              if (prop.value.type === 'Literal' && /#[0-9a-fA-F]+|rgb\(/.test(prop.value.value)) {
                context.report({ node: prop, messageId: 'noHardcodedColor' });
              }
            }
          });
        }
      },
    };
  },
};
```

## 7. Шкала токенов — справочник

### Отступы

| Токен | Значение | Использование |
|-------|----------|--------------|
| `--spacing-2xs` | 2px | Минимальный зазор между иконкой и текстом |
| `--spacing-xs` | 4px | Внутренний отступ маленьких элементов |
| `--spacing-sm` | 8px | Отступ кнопок, инпутов |
| `--spacing-md` | 12px | Отступ карточек, панелей |
| `--spacing-lg` | 16px | Основной отступ контейнеров |
| `--spacing-xl` | 20px | Отступ между секциями |
| `--spacing-2xl` | 24px | Отступ между крупными блоками |
| `--spacing-3xl` | 32px | Отступ между страницами/разделами |
| `--spacing-4xl` | 40px | Большие промежутки |
| `--spacing-5xl` | 48px | Очень большие промежутки |
| `--spacing-6xl` | 64px | Максимальный промежуток |

### Border-radius

| Токен | Значение | Использование |
|-------|----------|--------------|
| `--radius-sm` | 4px | Кнопки, инпуты, бейджи |
| `--radius-md` | 8px | Карточки, панели, выпадающие списки |
| `--radius-lg` | 12px | Модальные окна, диалоги, попапы |
| `--radius-xl` | 16px | Большие контейнеры, сайдбары |
| `--radius-full` | 9999px | Пилюли, аватары, круглые элементы |

### Тени

| Токен | Использование |
|-------|--------------|
| `--shadow-sm` | Лёгкая тень для приподнятых элементов |
| `--shadow-md` | Тень для карточек, панелей |
| `--shadow-lg` | Тень для модальных окон |
| `--shadow-popup` | Тень для попапов, выпадающих списков |
| `--shadow-floating` | Тень для плавающих элементов (тултипы, уведомления) |
